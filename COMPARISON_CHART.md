## Migration Guide

### From Bartender 4 to DivinicalUI:
1. Take screenshots of your Bartender layout
2. Install DivinicalUI
3. Run the setup wizard (choose your role)
4. Use the visual editor to match your previous layout
5. Export your profile for backup

**Time Required**: 10-15 minutes

### From ElvUI to DivinicalUI:
1. Export your ElvUI profile (backup)
2. Note your key customizations
3. Install DivinicalUI (can run alongside ElvUI temporarily)
4. Run the setup wizard
5. Manually recreate key customizations (unit frame positions, etc.)
6. Test thoroughly before disabling ElvUI

**Time Required**: 30-45 minutes
